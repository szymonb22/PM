# PM
Prosta aplikacja do zarządzania produkatmi  wykonana w .net 9

🚀 Funkcjonalności
•	Pobieranie listy produktów
•	Pobieranie produktu po ID
•	Dodawanie nowego produktu
•	Edycja produktu
•	Usuwanie produktu
🧱 Technologie
•	.NET 9 / ASP.NET Core Web API
•	Entity Framework Core
•	InMemory Database
•	xUnit (testy)
